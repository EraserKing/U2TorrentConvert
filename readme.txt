U2 Torrent Convert

Usage:

1. Install .Net Core 3.1 runtime (.NET Runtime 3.1.x) and install
https://dotnet.microsoft.com/download/dotnet-core/3.1

2. Create an folder called "Origin" under the application folder.

3. Copy all the torrent files of U2 to the "Origin" folder.

4. Edit the apikey.txt, paste the URL of your API inside and save.
https://u2.dmhy.org/privatetorrents.php

5. Run the app
Windows: run U2TorrentConvert.exe
Other: "dotnet U2TorrentConvert.dll"

6. Check the log.

7. Get the converted torrent files under "New" folder.
